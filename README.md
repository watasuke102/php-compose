# php-compose

PHP developing environment on Docker Compose

## How to use

1. Clone this repo
2. `docker compose up`
3. Access `localhost:8080`
4. Edit `src/index.php`

You can add other file like `name.php` to `src/` directory and see on `localhost:8080/name.php`.
